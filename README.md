# Golang invoice generator

A super fast golang package to generate invoices, delivery notes and quotations as pdf
using https://github.com/go-pdf/fpdf.

## Features

- Generate PDF invoices, delivery notes, and quotations
- Support for Code 128 barcodes
- Customizable styling and colors
- Multi-language support
- Automatic calculations (tax, discounts, totals)
- Logo support
- Header and footer customization

## Download from Github

```
go get -u github.com/angelodlfrtr/go-invoice-generator
```

## Quick start

```go
package main

import (
	"io/ioutil"
	"testing"

	generator "github.com/angelodlfrtr/go-invoice-generator"
)
```

### With Barcode Support

```go
// Create document with barcode
doc, _ := generator.New(generator.Invoice, &generator.Options{
	TextTypeInvoice: "FACTURE",
	AutoPrint:       true,
})

// Set barcode content (Code 128 format)
doc.SetBarCode("INV-2024-001")

// Build and save
pdf, _ := doc.Build()
pdf.OutputFileAndClose("invoice_with_barcode.pdf")
```
